# Hill Climb ThirdWeb

A web3 clone of Hill Climb powered by Unity, ThirdWeb GamingKit and ContractKit.

## Features

- Four levels.
- Four cars.

## Requirements

- Unity version `2021.3.15f1`.

## Build

- Create a new unity project.
- Import `Assets`, `packages` and `ProjectSettings` folders into your project.
